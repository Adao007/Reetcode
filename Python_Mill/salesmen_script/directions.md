# Best Salesmen Script

## Directions 
- You are given three csv files, use the three files and python to output the following:
1. Name of Salesmen
2. Location employeed 
3. Name of manager 
4. Annual Goal for Salesmen 
5. Percentage of Annual goal achieved
6. Output the list sorted by employees with the highest percentage. 


### Notes 
- if Manager column is missing, assume they are a manager. Thus they have no
manager. 
- Sales count if they are closed. Any other status are not counted toward the final percentage.
